# Support

Yapper is open source and maintained on a best-effort basis.

## Getting Help

- Read the [README](README.md) for install and build instructions.
- Read the [User Guide](docs/user-guide.md) for normal usage.
- Search existing GitHub issues before opening a new one.
- Include your macOS version, app version or commit, transcription engine, selected model, and steps to reproduce.

## What To Expect

Community help is welcome, but response times are not guaranteed. Pull requests with focused fixes are usually easier to review than broad rewrites.
